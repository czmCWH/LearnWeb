# ESLint + Prettier + Stylelint

## ESLint

```shell
$ npm install eslint --save-dev
```

## Prettier 代码格式化

```shell
$ npm install --save-dev --save-exact prettier
```
创建 `.prettierrc.json`，添加内容如下：

```json
{
  "tabWidth": 2,
  "useTabs": true,
  "printWidth": 80,
  "semi": true,
  "trailingComma": "es5",
  "singleQuote": true,
  "endOfLine": "lf"
}
```
