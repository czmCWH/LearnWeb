# 四、屏幕适配插件

* 1、postcss-pxtorem 插件

```shell
$ npm install -D postcss postcss-pxtorem
```

* 2、postcss-px-to-viewport-8-plugin 插件 -> 当前项目使用

```shell
$ npm install postcss-px-to-viewport-8-plugin -D
```

* 3、postcss-pxtorem（将 px 转换为 rem）配合 amfe-flexible（动态设置根字体大小）

```shell
# 安装移动端自适应库（运行时依赖）
$ npm install amfe-flexible

# 安装 PostCSS 插件（开发时依赖）
$ npm install postcss-pxtorem -D
```
