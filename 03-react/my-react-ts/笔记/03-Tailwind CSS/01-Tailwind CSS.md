### 1、主流CSS样式实现方案：
- 手写原生 CSS & CSS 预处理器（Sass/Less）；
- CSS-in-JS，如：Styled-components 停止维护、Emotion；
- Utility-First CSS，原子化CSS，如：Tailwind、UnoCSS


### 2、原子化CSS
原子化CSS：定义一些细粒度的 class，叫做原子 class，然后在 html 里直接引入这些原子化的 class。
