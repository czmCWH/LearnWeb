# 安装 html 模板插件

<https://github.com/vbenjs/vite-plugin-html/blob/main/README.zh_CN.md>

```shell
npm i vite-plugin-html -D

# 这样即可在 index.html 文件中可以使用环境变量
# <%=环境变量名%> 
```
