# 二、Ai 类目申请处理办法
## 1、微信小程序申请【深度合成-AI问答】类目需要的资料：

```
深度合成-AI问答，所需资质：
1、使用自研技术，提供：
小程序主体的《互联网信息服务算法备案》（算法类型为“生成合成类”）（注：点击可查看资质示例）

2、使用第三方技术（2选1）：
2.1 同时提供：
① 技术主体的《互联网信息服务算法备案》（算法类型为“生成合成类（深度合成）”）
②小程序主体与技术主体的合作协议（协议须含【算法名称】或【算法应用场景】或【备案编号】相关内容）（注：点击可查看资质示例）

2.2 深度合成服务《在用证明》 (证明须含【小程序主体】 、【小程序appid】 、【订单有效期】、【算法备案号】等内容）
```

##  2、由于资质1 和 资质2.1 很难实现，我们使用的是 “资质2.2” 方式来处理。处理步骤如下：

- 步骤1，先阅读文档：小程序深度合成服务在用证明获取指引：
<https://developers.weixin.qq.com/community/develop/doc/0006c8d1f5c298fd5e13bd2856b009>
- 步骤2，进入【微信服务市场】> 【深度合成】 > 购买一个免费的大模型
<https://fuwu.weixin.qq.com/search?tab=3&category=&serviceType=4-3-9&industry=&scene=&type=&label=84>
- 步骤3，购买成功后，把购买详情截图拼图，然后在类目申请页面上传即可；

需要注意的是，咨询微信客服，如果使用 2.2的【在用证明】则使用的大模型必须是 “微信服务市场” 中提供的。比如如果是 “豆包大模型” 则需要提供择2.1资质。我们虽然用的豆包，但是按2.2的方式解决了。

# 三、微信认证

## 【微信认证】有如下5步：
同意协议 > 填写资料 > 确认名称 > 填写发票 > 支付费用。

## 【确认名称】的相关解释：即确认小程序最终的名称。
- 填写申请认证的昵称，认证成功后会展现在小程序前端，认证审核成功（资质和名称审核）账号资料页显示认证标识(蓝V)。
- 境内账号每个自然年可以修改2次名称，如果2次机会用完了，可以通过微信认证流程改名，申请认证过程中可以修改一次名称，且认证成功后账号改名次数会重置为2次。每年1月1号账号改名次数都会重置为2次。

## 此操作的作用：
- 1、绑定小程序账号的微信账号；
- 2、缺认小程序的名称；
- 3、购买小程序，需付款300元一年；
