## 3、查询相机相册授权

<https://mp.weixin.qq.com/s/Fdv-0D5ybR3T3PIhr6gBJA>，小程序 — canvas图片合成

```js
save() {
  let self = this;
  if (!this.imgSrc) {
    wepy.showToast({ title: '请先选择图片', icon: 'none', duration: 2000 });
    return false;
  }
  wx.getSetting({
    success(res) {
      if (!res.authSetting['scope.writePhotosAlbum']) {
        wx.authorize({
          scope: 'scope.writePhotosAlbum',
          success() {
            self.saveImage();
          }
        });
      } else {
        self.saveImage();
      }
    }
  });
}
```
