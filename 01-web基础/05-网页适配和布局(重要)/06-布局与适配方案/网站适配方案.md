# 网站适配方案

## 1、Masonry.js 瀑布流布局网站

<https://masonry.desandro.com/>
<https://github.com/desandro/masonry>

Masonry 是一个 JavaScript 网格布局库。它的工作原理是根据可用的垂直空间将元素放置在最佳位置，有点像泥瓦匠在墙上安装石头。我们在互联网上也许看到过很多瀑布流的案例.

```shell
$ npm install masonry-layout
```
